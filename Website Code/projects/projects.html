Wildfire Risk Prediction & Visualization System

Role: ML Engineer | Data Science Team

Tech: Python, XGBoost/Random Forest, scikit-learn, Flask/FastAPI, PostgreSQL, PostGIS, Leaflet.js, SHAP

A web-based system that predicts and visualizes wildfire risk for California regions using machine learning. Responsible for building the data ingestion pipeline pulling from USGS, NOAA, and NASA FIRMS datasets, preprocessing spatial data (CRS alignment, resolution matching), and training ML models to generate daily/weekly risk predictions. Features interactive risk heatmaps, evacuation routes, and real-time alerts. Currently in Sprint 1 - setting up API integrations and data validation framework.
