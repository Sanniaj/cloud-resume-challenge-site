WHO AM I?

Hi, I'm Sannia Jean, a Computer Science student at California State University, Northridge (graduating May 2026). I'm passionate about building practical tech that works with real-world data. Recently that's meant training ML models, engineering data pipelines, and setting up the cloud infrastructure to run them. What keeps me engaged is being close to the foundation of how a system actually works.

WHAT I'M WORKING ON

  • FireScope. I worked on the ML and data ingestion for a team-built platform predicting California wildfire risk. I built the feature engineering that turned raw NASA FIRMS, MODIS EVI, and weather feeds into inputs for a scikit-learn Random Forest, served through a Flask inference API.
  • This site. Built end-to-end on AWS with Terraform: private S3 behind CloudFront with Origin Access Control, a Python Lambda + API Gateway + DynamoDB visitor counter, and a GitHub Actions deploy pipeline.
  • PocketPilot. I redesigned the web front-end of a team AI finance app in React + TypeScript + Tailwind, adapting a Figma-designed component library into a cohesive visual system.
